# Crescent Credit Union Staff Directory

## Overview
This repository contains an employee directory search application developed as part of my coursework while pursuing a Bachelor of Business Administration in Information Systems and Technology (IST).

The application allows users to search and filter employee records based on last name, department, and job position. The project demonstrates advanced JavaScript programming concepts including object-oriented programming, JSON data processing, dynamic table generation, search filtering, regular expressions, sorting algorithms, and DOM manipulation.

## Features
* Employee directory search system
* Dynamic staff record filtering
* Search by last name
* Search by department
* Search by position title
* Multiple search modes (Contains, Begins With, Exact Match)
* Dynamic table generation
* Employee contact information display
* Automatic result sorting by employee ID
